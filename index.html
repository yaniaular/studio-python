Listas y Tuplas
  • Listas son mutables, las tuplas no
  • Para declarar listas se usan corchetes
  • Para declarar tuplas, se usan paréntesis
  • Tanto listas como tuplas pueden tener elementos diversos
  • Las listas permiten agregar y borrar elementos
  • Las tuplas no permiten agregar o borrar elementos
  • En listas y tuplas se acceden a sus elementos usando índices positivos y negativos entre corchetes
  • Para obtener un slice, se utilizan los ":"

for..in
 En python, podemos recorrer objetos iterables con for. Por ejemplo: listas, tuplas, diccionarios, xrange() o conjuntos. 

        >> lista = [11, 4.5, 'DB', 'python']
        >> diccionario = {'curso':'base de datos', \
        'taller':'python','laboratorio':6}
        >> for i in lista:
                print i
        >> for i, j in diccionario.items():
                print i, j
        >> for i, j in enumerate(lista):
                print i, j
        >> for i in (1,2,3):
                print i
                break
range() y xrange()
  • La función range() genera una lista la cual es manejada elemento por elemento.
  • La función xrange() produce un objeto iterable a través del cual se puede pasar sin el peligro de lo que podría ser un objeto (lista) muy grande.
  • for x in range(10000), genera una lista
  • for x in xrange(10000), genera un iterable

        >> type(range(100))
        >> type(xrange(100))
